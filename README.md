# giftChatCounter2

카톡 단톡방의 대화량과 기프티콘 선착순 선물하기를 쏘고 받은 횟수를 세어줍니다.

대화기록 텍스트파일의 이름을 KakaoTalkChats.txt로 저장하고 실행하세요. (채팅방 설정 - 대화 내용 내보내기 - 텍스트만 보내기)

주의: 대화 내용 내보내기를 할 때 카톡의 언어설정이 한국어여야 합니다.
